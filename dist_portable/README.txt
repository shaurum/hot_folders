# 🖨️ HotFolders

Автоматическая обработка файлов в типографии.

## 🚀 Запуск

1. Распакуйте архив
2. Запустите **`Start.bat`**
3. Иконка появится в трее

## ⚙️ Настройка

1. Двойной клик по иконке в трее
2. Укажите путь к ImpositionWizard
3. Добавьте папки (A4, A5...)
4. Настройте пресеты

## 📁 Файлы

- **Start.bat** — запустить
- **Stop.bat** — остановить
- **config.json** — настройки

## 📞 Помощь

При проблемах проверьте:
1. Запущено ли приложение (смотрите трей)
2. Правильный ли путь к ImpositionWizard
3. Логи в файле `hot_folders.log`
