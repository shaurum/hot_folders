# Hot Folders

Приложение для автоматической обработки файлов в Windows. Мониторит папки, конвертирует изображения в PDF и применяет ImpositionWizard.

## Возможности

- 📁 **Мониторинг папок** — автоматическое отслеживание новых файлов в указанных папках
- 🖼️ **Конвертация изображений** — поддержка TIF, TIFF, JPG, PNG, BMP, GIF, WEBP → PDF
- 📄 **Imposition** — применение пресетов ImpositionWizard к PDF файлам
- 🔔 **Системный трей** — работа в фоне с иконкой в трее
- ⚙️ **Настройки** — гибкая настройка папок и пресетов через GUI
- 🚀 **Автозапуск** — опциональный запуск при старте Windows

## Установка

### 1. Требования

- Python 3.8 или выше
- Windows 10/11
- ImpositionWizard (установлен и настроен)

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск приложения

```bash
python main.py
```

После запуска в системном трее появится иконка приложения.

## Использование

### Настройка папок

1. Кликните дважды по иконке в трее (или правой кнопкой → Настройки)
2. Перейдите на вкладку **"Папки"**
3. Нажмите **"Добавить"**
4. Заполните поля:
   - **Название** — идентификатор папки (например: A4, A5, единички)
   - **Входная папка** — папка для мониторинга
   - **Выходная папка** — куда сохранять готовые PDF
   - **Пресет Imposition** — имя пресета в ImpositionWizard
5. Нажмите **OK**

### Настройка ImpositionWizard

1. В настройках перейдите на вкладку **"Общие"**
2. Укажите путь к `ImpositionWizard.exe` если он не в PATH
3. При необходимости включите **"Запускать при старте Windows"**

## Как это работает

1. Приложение мониторит указанные папки
2. При появлении нового файла:
   - Если это изображение (TIF, JPG, PNG и т.д.) → конвертирует в PDF
   - Если это PDF → использует как есть
3. Вызывает ImpositionWizard с указанным пресетом
4. Сохраняет результат в выходную папку
5. Показывает уведомление об успехе или ошибке

## Поддерживаемые форматы

- **Изображения**: `.tif`, `.tiff`, `.jpg`, `.jpeg`, `.png`, `.bmp`, `.gif`, `.webp`
- **PDF**: `.pdf`

## Решение проблем

### ImpositionWizard не найден
Убедитесь, что путь к ImpositionWizard указан правильно в настройках.

### Файлы не обрабатываются
- Проверьте, что папка включена в настройках (галочка "Включить мониторинг")
- Убедитесь, что формат файла поддерживается
- Проверьте логи ошибок в уведомлениях

### Ошибка автозапуска
Запустите приложение от имени администратора для настройки реестра.
